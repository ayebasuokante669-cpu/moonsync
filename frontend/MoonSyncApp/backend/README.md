# MoonSync Backend

This repository contains the backend services for **MoonSync**, a menstrual and reproductive health tracking application.

## Overview
The backend is responsible for:
- User authentication and authorization
- Menstrual cycle tracking logic
- AI-powered chat assistance
- Voice input and audio responses
- Notifications and reminders
- Community features (posts, comments)
- Database interactions

## Tech Stack
- **Python**
- **FastAPI**
- **Supabase (PostgreSQL)**
- **Firebase Auth (Google Sign-In)**
- **PyTorch / Transformers (LLM integration)**
- **Whisper (Speech-to-Text)**
- **TTS service (Text-to-Speech)**
