# MoonSync Frontend

This folder contains the frontend application for **MoonSync**, a menstrual and reproductive health tracking app.

## Overview
The frontend is responsible for:
- User onboarding and splash screens
- Cycle tracking UI
- Community feed and interactions
- AI chat interface
- Voice input and audio responses
- Notifications and reminders UI
- Settings and personalization

## Tech Stack
- **Flutter** (Android / iOS)
- **Firebase Auth (Google Sign-In)**
- **REST API integration** (Backend)
- **Local storage** (temporary caching)

## Key Features
- Clean, calming UI focused on health and comfort
- Accessibility-first design
- Smooth animations and transitions
- Secure authentication flow
- Offline-friendly experience where possible

## Notes
- The frontend communicates with the backend via REST APIs.
- Authentication is handled via Firebase and linked to backend user records.
- Design decisions prioritize user privacy and emotional safety.
